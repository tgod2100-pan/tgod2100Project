## 目錄與路徑操作

```cmd
cd                   # 顯示目前所在目錄
cd <資料夾名稱>       # 進入指定資料夾
cd ..                # 返回上一層目錄
cd \                 # 回到目前磁碟槽根目錄
D:                   # 切換到 D 磁碟槽（磁碟機切換必用此方式）

檔案與資料夾操作
dir                  # 顯示目錄內檔案和資料夾列表
mkdir <資料夾名稱>    # 建立新資料夾
rmdir <資料夾名稱> /s /q  # 刪除資料夾及其內容（小心使用）
del <檔案名稱>        # 刪除指定檔案
copy <來源> <目的地>   # 複製檔案
move <來源> <目的地>   # 移動或重新命名檔案/資料夾

磁碟機切換
C:                   # 切換到 C 磁碟槽
D:                   # 切換到 D 磁碟槽

系統資訊
systeminfo           # 顯示系統詳細資訊
tasklist             # 顯示目前執行中的程序清單
ipconfig             # 查看本機 IP 配置

其他實用指令
cls                  # 清除螢幕畫面
echo Hello World     # 在畫面輸出文字
pause                # 暫停執行並顯示「Press any key to continue...」
exit                 # 離開 CMD 視窗
